¿QUÉ ES UN CONJUNTO NUMERICO?

Son conjuntos de números. En su forma más genérica se refiere a los grandes conjuntos de números como: naturales, enteros, fraccionarios, racionales, irracionales, reales, imaginarios y complejos.

Operaciones

Unión o reunión de conjuntos.

Es la operación que nos permite unir dos o más conjuntos para formar otro conjunto que contendrá a todos los elementos que queremos unir pero sin que se repitan.

El símbolo que se usa para indicar la operación de unión es el siguiente: ∪.

Refuerza el Conocimiento!
Intersección de conjuntos.

Es la operación que nos permite formar un conjunto, sólo con los elementos comunes involucrados en la operación y los elementos no comunes será excluido.

El símbolo que se usa para indicar la operación de intersección es el siguiente: ∩.

Refuerza el Conocimiento!
Complemento de un conjunto.

Es la operación que nos permite formar un conjunto con todos los elementos del conjunto de referencia o universal, que no están en el conjunto.

En esta operación el complemento de un conjunto se denota con un apostrofe, algo como esto A'.

Refuerza el Conocimiento!
Diferencia de conjuntos.

Es la operación que nos permite formar un conjunto, en donde de dos conjuntos el conjunto resultante es el que tendrá todos los elementos que pertenecen al primero pero no al segundo.

El símbolo que se usa para esta operación es el mismo que se usa para la resta o sustracción, que es el siguiente: -.

Tipos de Conjuntos

Conjunto vacío

Se trata de un conjunto que no tiene ningún elemento. Se representa mediante el símbolo Ø o con dos llaves vacías {}

    como se puede deducir, ningún elemento del universo puede constituir este conjunto. , dado que de constituirlo deja automáticamente de ser un conjunto vacío.
    | Ø | = 0 y X ∉ Ø, no importa lo que X pueda ser.

Refuerza el Conocimiento!
Conjuntos iguales

Dos conjuntos son iguales en el caso de que contengan los mismos elementos.

    Por ejemplo:
    A = {números impares del 1 al 15} y B = {1,3,5,7,9,11,13,15}
    Entonces A = B.
    Si no tienen los mismos elementos C = {1,2,3} y D = {2,3,4}, se dice que son diferentes por tanto C ≠ D.

Refuerza el Conocimiento!
Conjuntos unitarios

se distinguen por tener solo un elemento. No importa qué tipo de elemento tenga el conjunto, si tiene un solo elemento es llamado conjunto unitario.

    Por ejemplo, el conjunto { 0 } es un conjunto unitario.
    Observe que un conjunto como por ejemplo,
    { { 1; 1; 1 } } es también un conjunto unitario.

Conjuntos finitos

Los conjuntos finitos son aquellos en los que es posible contar todos sus elementos.

    Por ejemplo,
    {Las vocales} = {a, e, i, o, u}

Conjuntos infinitos

Se trata de lo contrario a los conjuntos finitos. En los conjuntos infinitos hay infinidad de elementos

    Por ejemplo {los números} = {1,2,3,4,5,6,7,8,...}

Subconjuntos

Como su propio nombre indica, se trata de conjuntos dentro de conjuntos con más elementos.

    Por ejemplo, un casco de seguridad es parte de los diferentes tipos de cascos, por este motivo diríamos que el conjunto de cascos de seguridad es un subconjunto del conjunto de todos los cascos.
    Así pues: C = {Cascos de seguridad} y H = {todos los cascos},
    entonces C ⊂ H.
    Esta expresión se lee como C es un subconjunto de H.
